pretext-powered fake monospace

Every font on this page is being coerced into equal-width grapheme cells.

The font family stays proportional, but each rendered grapheme gets the same measured advance width. That makes serif, sans, and display faces feel like strange custom monospace cuts.

Optical mode uses softer cell widths, centers glyphs in each cell, and trims spaces so paragraphs stay readable.

Editorial serif

Typography can be made rigid without changing the typeface itself.

This demo uses measured grapheme widths and then expands every grapheme to a shared cell width for that font. The result is not true font engineering. It is a browser-side illusion.
